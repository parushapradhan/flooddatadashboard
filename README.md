```bash
git clone https://github.com/richard-unterberg/next-leaflet-starter-typescript
```

### <a id="start-up"></a> 🏍️ Start up

```bash
npm run dev
# or
yarn dev
```

```bash
npm run build
# or
yarn build
```

Start build locally

```bash
npm run start
# or
yarn start
```
